Activity per month and new problems

With the line chart we visualize how much the user activity on the website has increased from 2009 until 2016 while showing how many new problems were created every month. In our case, activity for a month is the sum of the unique visits of users each day of the specific month. We count one visit for a user if there is at least one solution submission by him.

It is obvious that as the time passed the number of users got bigger, thus the site gained popularity. Starting with under 200 users, there was a steady increase each year with few and small "downs". From May 2013 where visitors surpassed 1,000, activity started blooming, touching almost 6000 visitor on August 2015. This could have happened because of the manh new problems that were added the past 3-4 months. If we notice the peaks of both timelines, there is a good chance that a corellation exist between these 2. activity.
